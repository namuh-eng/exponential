# Public pricing packaging

Exponential public pricing uses six plans:

- **Community Self-hosted** — $0, free forever for core self-host/community issue tracking.
- **Cloud Free** — $0 hosted entry plan for individuals and small teams.
- **Cloud Team** — **$7/user/mo billed annually** or **$9/user/mo monthly** for managed cloud collaboration and hosted scale.
- **Cloud Business** — **$12/user/mo billed annually** or **$15/user/mo monthly** for managed cloud scale, admin controls, security workflows, integrations, analytics, and priority support.
- **Enterprise Cloud** — custom annual contract for managed cloud scale, SAML/SCIM, audit exports, compliance support, security review support, and dedicated commercial support.
- **Enterprise Self-hosted** — custom annual contract for organizations that need to operate the product themselves with commercial license terms, deployment guidance, upgrade planning, compliance support, and priority support.

Core self-host/community issue tracking remains free. Paid gates should be framed around managed cloud hosting, scale, advanced admin/security/compliance requirements, and support rather than crippling basic issue tracking.

Public CTAs must not embed private Stripe IDs, credentials, personal emails, or secrets.
